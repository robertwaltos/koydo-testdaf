# koydo-testdaf

TestDaF exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM025 |
| Name | TestDaF |
| Country | Global |
| Category | Language proficiency |
| Tier | Tier 4 |
| Wave | Wave 6 |

This app is a study aid based on publicly available syllabus and exam patterns. Not affiliated with any official examination authority.
